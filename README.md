## Cafe Social – Location-Locked Social Gaming for Cafés

Cafe Social is a **location-locked social gaming platform for physical venues**, starting with coffee shops.

- Users play a **simple 2–4 player multiplayer mini-game** that only works when they are inside a partner café (geofenced).
- They earn **XP and loyalty rewards** for each venue and for their global profile (free coffee, dessert, cross-café achievements, etc.).
- Café owners get a **dashboard and loyalty system disguised as a game**, with basic analytics (unique/returning visitors, sessions, group play, rewards).

Long term, the same platform can extend from cafés to **bars, events, malls, airports, and other dwell-time venues**.

This folder will contain the new implementation for Cafe Social, reusing patterns and components from the existing `frugends-app` (Expo) and `cheeroes-backend` (NestJS + Prisma) projects where possible.
